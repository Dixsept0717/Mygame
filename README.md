# 🐟 FishGame 大鱼吃小鱼 - Qt 游戏项目
## 📌 版本：v3.0（完整版更新）

基于 Qt 6 + QGraphicsView 框架开发的休闲大鱼吃小鱼游戏，支持完整游戏流程、难度选择、主题切换、BGM 音效系统与真实贴图素材，可直接打包为独立 `.exe` 运行。

## 🛠 技术栈
- 开发语言：C++
- 核心框架：Qt 6（`QGraphicsView`/`QGraphicsScene`/`QMediaPlayer`）
- 图形系统：`QGraphicsPixmapItem` + 主题贴图素材
- 音频系统：`QMediaPlayer` 实现 BGM 与音效
- 构建工具：qmake
- 支持平台：Windows（可打包为独立应用）

## 🎮 游戏玩法说明
1. 打开游戏 → 主菜单选择「开始游戏」
2. 选择难度（简单/普通/困难），进入游戏
3. 控制玩家鱼移动，吃掉比自己小的鱼，躲避大鱼
4. 获得经验升级，鱼的体型会随等级变大
5. 达到 **10 级** 触发胜利条件，游戏通关
6. 被更大的鱼碰撞 → 触发游戏失败，进入结算界面
7. 游戏中按 `ESC` 呼出暂停菜单，支持设置、重新开始、返回主菜单

## 🎚 难度系统
难度影响鱼的刷新频率、数量和移动速度，适配不同游戏节奏：
- **简单**：刷鱼间隔长、数量少、移动速度慢，适合新手入门
- **普通**：标准节奏，刷鱼/速度平衡，体验完整玩法
- **困难**：刷鱼间隔短、数量多、移动速度快，挑战操作上限

## 🚀 如何运行
### 环境要求
- Qt 6.x（推荐 6.11.0）
- MinGW 编译器（Windows 平台）

### 运行步骤
1. 克隆项目到本地
   ```bash
   git clone https://github.com/Dixsept0717/Mygame.git
   ```
2. 用 Qt Creator 打开项目文件 `Mygame.pro`
3. 配置对应 Qt 编译套件（如 `Desktop Qt 6.11.0 MinGW 64-bit`）
4. 构建并运行项目，即可进入游戏

### 打包为独立应用
1. 切换构建模式为 `Release`，编译生成 `.exe` 文件
2. 使用 Qt 官方工具 `windeployqt` 打包依赖库
3. 打包完成后，文件夹内的 `.exe` 可直接双击运行，无需安装 Qt
